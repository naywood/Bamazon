# Bamazon
Fake Amazon using MySQL, Node, Inquirer, and JavaScript. This version "sells" gardening/plant supplies. This app makes a simple command line based storefront.
<br>
## To run
* git clone https://github.com/naywood/Bamazon.git
* cd bamazon
* npm install
* node bamazon.js
<br>
